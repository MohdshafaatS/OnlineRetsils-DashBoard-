
🛍Online  Retail Sales Dashboard | Power BI Project



Excited to share my latest Power BI project where I analysed retail transaction data and transformed raw data into meaningful business insights 📊



This project focuses on tracking sales trends, understanding country-level performance, and building a structured data model for accurate reporting.



📊 Dashboard Highlights:



🔹 Total Sales: 200K+

🔹 Countries Analysed: United Kingdom & Others

🔹 Products Tracked: Multiple StockCodes

🔹 Time Analysis: Year–Month–Day hierarchy

🔹 Data Model: Structured Relationships Between Tables



📈 Key Insights:



✔ Sales trends vary significantly across different time periods.

✔ The United Kingdom contributes the highest share of sales.

✔ Product-level tracking helps identify high-value stock items.

✔ Date hierarchy enables deep time-based analysis.

✔ A well-structured relationship model ensures accurate reporting and faster dashboard performance.



🎯 What This Dashboard Delivers:



• Sales trend analysis using Invoice Date

• Country-wise sales performance tracking

• Product-level sales breakdown

• Interactive filtering using Date hierarchy

• Strong data modelling through relationship mapping
